# Brain-Tumor-MRI-Classification-SSL
Brain tumor MRI classification using self-supervised SimCLR pretraining with EfficientNetB3 encoder, followed by supervised fine-tuning. Includes advanced brain-region cropping, strong augmentation, contrastive NT-Xent loss, and Grad-CAM visualization. Achieved 98.32% test accuracy on glioma, meningioma, pituitary &amp; no-tumor classes.
